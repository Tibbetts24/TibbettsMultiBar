# Tibbetts MultiBar

Tibbetts MultiBar replaces the default XP and Reputation bars with a fully
customizable alternative.

## Features
- Custom textures and fonts
- Optional tick marks (10% / 20%)
- Hover tooltips showing tick percentages
- Blizzard-style color pickers with opacity
- Works on Retail, Classic Era, TBC Anniversary, and Wrath

## Commands
- /tmb
- /tibbettsmultibar

## Notes
Designed to be cross-client safe. No SharedXML dependencies.
